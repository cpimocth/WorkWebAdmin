Consumer Price Index Reporting System

ระบบรายงานสถิติดัชนีราคาผู้บริโภค (CPI)

กรุณาเลือกส่วนงานที่ท่านรับผิดชอบ เพื่อเข้าสู่เมนูรายงานและสรุปผลข้อมูล สำหรับการใช้งานในส่วนบริหาร ระบบจะกำหนดให้มีการยืนยันสิทธิ์ก่อนเข้าใช้งาน

🛡️

ส่วนงานบริหาร (Admin)

สำหรับผู้มีสิทธิ์เข้าถึงรายงานสรุปภาพรวมและข้อมูลชุดนอกเขต

📊 สรุปผลภาพรวมจังหวัด → 📈 สรุปผลชุดนอกเขต → 📈 CPI Dashboard Contribution Report →
หมายเหตุ: เมนูนี้สงวนสิทธิ์สำหรับเจ้าหน้าที่ส่วนงานบริหารที่ได้รับรหัสผ่านสำหรับเข้าใช้งานระบบเท่านั้น
สำนัก/หน่วยงานที่รับผิดชอบระบบรายงานข้อมูล CPI
